# Spitfire Product Landing Page Demo

This is a premium product landing page demo showcasing high-performance scroll animations and a modern, high-end aesthetic.

## 🚀 Overview

This site is a demonstration of advanced web development techniques, featuring:
- **Cinematic Scroll Animation**: A custom canvas-based frame animation engine that responds to user scrolling.
- **Dynamic Text Overlays**: Smooth, progress-based text transitions that highlight product features.
- **Premium Design System**: A sleek, dark-mode focused UI with vibrant accent colors and glassmorphism.
- **Responsive Animations**: Tailored animation scaling to ensure a great experience on both mobile and desktop.

## 🛠️ Skill Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animation Engine**: [Framer Motion](https://www.framer.com/motion/)
- **Rendering**: HTML5 Canvas API
- **Icons**: [Lucide React](https://lucide.dev/)

## 📦 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
